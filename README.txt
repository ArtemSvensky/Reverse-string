
Как проверить работоспособность программы:

Необходимо запустить приложение и ввести строку. Важно! Данное приложение поддерживает только латиницу. 


Описание:

Класс String представляет собой реализацию динамических строк на языке C++11 без использования сторонних библиотек или контейнеров STL. Он поддерживает строки произвольного размера и использует базовые операции работы с динамической памятью (функции malloc()/free() или операторы new/delete) для манипуляции последовательностями символов типа char.


Особенности:

Класс String не является шаблонным.
Поддерживается конструктор копирования, оператор присваивания и move-семантика.
Есть оператор +=, который позволяет добавлять содержимое одной строки к другой.
Поддерживается внешний оператор сложения двух строк.
Операции могут быть применены как к объектам типа String, так и к традиционным С-строкам, оканчивающимся нулем.
